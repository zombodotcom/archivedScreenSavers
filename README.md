# aSS — Archived Screen Savers

A WebGL museum of classic screensavers from every era of computing. Relive the nostalgia of 3D Pipes, Flying Toasters, Flurry, and 120+ more effects spanning Windows, Mac, Linux, and beyond.

**[Live Demo](https://archivedsscreensavers.netlify.app/)** | **[Full Screensaver Archive](ARCHIVE-new.md)**

## Quick Start

```bash
# Just open it
open index.html

# Or serve it (some browsers restrict fullscreen from file://)
npm run dev
# -> http://localhost:5173
```

## Controls

| Key | Action |
|-----|--------|
| **Mouse move** | Show UI panel |
| **F** | Toggle fullscreen |
| **S** | Toggle shuffle mode |
| **Space** | Jump to random effect |
| **Escape** | Hide UI |
| **Click effect** | Play that effect |

## Contributing

Want to add a screensaver? Here's how:

### 1. Create the Effect

Add your effect to `src/js/effects.js`:

```javascript
register('myeffect', `
// Your GLSL fragment shader
void main() {
  vec2 uv = gl_FragCoord.xy / u_resolution;
  float t = u_time;

  // Your shader magic here
  vec3 col = vec3(uv.x, uv.y, sin(t) * 0.5 + 0.5);

  fragColor = vec4(col, 1.0);
}
`, { name: 'My Effect', desc: 'A brief description' });
```

### 2. Available Uniforms

Every shader gets these for free:

| Uniform | Type | Description |
|---------|------|-------------|
| `u_time` | `float` | Elapsed time in seconds |
| `u_resolution` | `vec2` | Canvas size in pixels |
| `u_mouse` | `vec2` | Mouse position (0-1, bottom-left origin) |

### 3. Helper Functions

These are available in all shaders:

```glsl
float hash(vec2 p)           // Random 0-1 from 2D input
float noise(vec2 p)          // Smooth noise
float fbm(vec2 p)            // Fractal brownian motion
float line(vec2 uv, vec2 a, vec2 b)  // Distance to line segment
mat2 rot(float a)            // 2D rotation matrix
```

### 4. Add to an Era

In `src/js/app.js`, add your effect ID to the appropriate era:

```javascript
const ERAS = {
  win95: {
    name: 'Windows 95/98',
    effects: ['pipes', 'maze', ..., 'myeffect']  // Add here
  },
  // ...
};
```

### 5. Adding Settings (Optional)

For configurable effects like 3D Pipes:

```javascript
register('myeffect', shaderCode, {
  name: 'My Effect',
  desc: 'Description',
  settings: {
    speed: { type: 'range', min: 0.1, max: 2, default: 1, label: 'Speed' },
    color: { type: 'select', options: ['rainbow', 'mono'], default: 'rainbow', label: 'Color' },
    glow: { type: 'checkbox', default: true, label: 'Enable Glow' }
  }
});
```

Settings are passed to your shader via `u_settings_*` uniforms.

### 6. Test & Submit

```bash
npm run dev   # Start dev server
# Test your effect, then submit a PR!
```

## Architecture

```
src/
├── js/
│   ├── ass.js      # Core engine (WebGL2, effects registry)
│   ├── effects.js  # All screensaver shaders (~120 effects)
│   └── app.js      # UI, eras config, settings panel
├── css/
│   └── styles.css  # Museum UI styling
└── index.html      # Entry point
```
